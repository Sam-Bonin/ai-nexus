# Quick Start Guide - AI Nexus

Get up and running with AI Nexus in 5 minutes.

## 1. Download & Install

**Download**: [AI Nexus v1.0.0](https://github.com/Sam-Bonin/ai-nexus/releases/download/v1.0.0/AI.Nexus-1.0.0.dmg) (102MB)

1. Double-click the DMG file
2. Drag **AI Nexus** to **Applications** folder
3. Eject the DMG

## 2. First Launch (Important!)

**‚ö†Ô∏è Don't double-click!** macOS will block unsigned apps.

Instead:
1. Go to **Applications** folder
2. **Right-click** on **AI Nexus**
3. Select **"Open"**
4. Click **"Open"** in the security dialog

After this one-time step, the app launches normally.

## 3. Get Your API Key

You need an OpenRouter API key (free):

1. Visit **https://openrouter.ai**
2. Sign up or log in
3. Go to **Settings** ‚Üí **API Keys**
4. Click **"Create Key"** and copy it

## 4. Configure AI Nexus

1. Launch AI Nexus
2. Click the **‚öôÔ∏è Settings** icon (top right)
3. Go to **"API Settings"**
4. Paste your API key
5. Click **"Save API Key"**

## 5. Start Chatting!

You're ready! Try these features:

### Send a Message
- Type in the input box at the bottom
- Press **Enter** or click **Send**
- Watch the response stream in real-time

### Select a Model
- Click the **model selector** (top of chat)
- Choose from 5 Claude models:
  - Claude Sonnet 4.5 (recommended)
  - Claude Sonnet 4
  - Claude 3.7
  - Claude 3.5
  - Claude Opus 4

### Attach Files
- Click the **üìé paperclip** icon in the input area
- Select an image or PDF
- AI will analyze the file in its response

### Enable Extended Thinking
- Toggle **üí≠ Extended Thinking** above the input
- See Claude's reasoning process separately
- Great for complex problems

### Create Projects
- Click **"+ New Project"** in the sidebar
- Organize conversations by topic
- AI automatically categorizes new chats

### Change Themes
- Click **‚öôÔ∏è Settings** ‚Üí **"Personalization"**
- Choose from 5 color palettes
- Switch between Light/Dark modes
